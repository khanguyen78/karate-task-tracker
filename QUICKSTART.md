# Quick Start Guide

## Setup (3 steps)

1. **Create folder structure:**
```bash
   mkdir -p karate-tracker/static
   mkdir -p karate-tracker/templates
   mkdir -p karate-tracker/data
```

2. **Save all files** to their locations shown above

3. **Run:**
```bash
   cd karate-tracker
   docker-compose up --build
```

4. **Open:** http://localhost:8000

## Features

✅ Countdown timer
✅ Pause/Resume button
✅ Auto-mode (Begin All Tasks)
✅ Clear All Tasks
✅ Reset Session
✅ Multi-user support
✅ Data persistence
✅ Progress tracking
