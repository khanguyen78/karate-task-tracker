📄 How to Create & Upload Your Task CSV

▼
  1. Open any spreadsheet app (Excel, Google Sheets, Numbers) or a plain text editor like Notepad.
  2. Add the header row exactly as: task, description, estimated_time
  3. Fill in one task per row. The estimated_time column is in minutes. Description is optional — leave it blank but keep the comma.
  4. Export the file as .csv:
    • Excel: File → Save As → CSV
    • Google Sheets: File → Download → CSV
    • Notepad: Save As → name the file tasks.csv
  5. Log in with your name, then click Upload Tasks 📤 on your dashboard and select the file.

📋 Format & Example

task,description,estimated_time Front Kick,Practice front kick technique,10 Roundhouse Kick,Focus on hip rotation,15 Kata Heian Shodan,Full kata run-throughs,20 Sparring Drills,Partner sparring combos,30 Cool Down Stretches,,5

⚠️ The header row must be included. If you skip the description, keep the comma: Cool Down,,5

💡 Each student's CSV is private — uploading only adds tasks to your own account, not anyone else's.

⚠️ Note that this application runs in a Docker container and currently in beta testing. Any information that you upload maybe lost on the next build & deploy. You may need to re-upload tasks at any point so please be sure to save a local copy.

🕹️ Button Guide

▼

📤 Uploading Tasks

  • Upload Tasks 📤 Shown when you have no tasks yet. Select your CSV file to load your task list.
  • + Add More Tasks Adds tasks from another CSV on top of your existing list — useful for adding new drills without losing your current ones.

▶️ Running Tasks

  • ▶️ Begin All Tasks Runs all tasks back-to-back automatically. Each task runs for its estimated time then advances to the next.
  • 🚀 Start Starts the timer for a single task. Only one task can run at a time.
  • ✅ Finished! Completes the active task, records your time, and calculates your focus and impact scores.
  • ❌ Cancel Task Stops the current task without saving any progress for it.

📊 Navigation

  • 📊 View Results Opens your results page showing all completed tasks, focus scores, impact scores, and total training time.
  • 🏠 Home Returns to this page so a different student can log in.

⚙️ Managing Tasks

  • ✏️ Edit Edit a task's title, description, or estimated time.
  • 🗑️ Clear All Tasks Permanently deletes all your tasks and completion history. Other students are unaffected. Cannot be undone.
  • 🔄 Reset Session Deletes all your completion records and resets your streak, but keeps your task list intact.

📈 Scores Explained

  • Focus Score How closely your actual time matched the estimated time. 1.0 is perfect; below 0.5 means significantly over or under.
  • Impact Score A combined score (0–10) based on task difficulty and your focus score. Higher is better.
