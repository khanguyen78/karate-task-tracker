📄 How to Create & Upload Your Task CSV

▼
  1. Open any spreadsheet app (Excel, Google Sheets, Numbers) or a plain text editor like Notepad.
  2. Add the header row exactly as: task, description, estimated_time, difficulty
  3. Fill in one task per row. The estimated_time column is in seconds. Description is optional. difficulty accepts: easy, medium, hard, expert — leave blank to auto-assign based on time.
  4. Export the file as .csv:
    • Excel: File → Save As → CSV
    • Google Sheets: File → Download → CSV
    • Notepad: Save As → name the file tasks.csv
  5. Log in with your name, then click Upload Tasks 📤 on your dashboard and select the file.

📋 Format & Example

task,description,estimated_time,difficulty Front Kick,Practice front kick technique,600,easy Roundhouse Kick,Focus on hip rotation,900,medium Kata Heian Shodan,Full kata run-throughs,1200,hard Sparring Drills,Partner sparring combos,1800,expert Cool Down Stretches,,300,

⚠️ The header row must be included. If you skip the description, keep the comma: Cool Down,,5

💡 Each student's CSV is private — uploading only adds tasks to your own account, not anyone else's.

🕹️ Button Guide

▼

📤 Uploading Tasks

  • Upload Tasks 📤 Shown when you have no tasks yet. Select your CSV file to load your task list.
  • + Add More Tasks Adds tasks from another CSV on top of your existing list — useful for adding new drills without losing your current ones.

▶️ Running Tasks

  • ▶️ Begin All Tasks Runs all tasks back-to-back automatically. Each task runs for its estimated time then advances to the next.
  • 🚀 Start Starts the timer for a single task. Only one task can run at a time.
  • ✅ Finished! Completes the active task, records your time, and calculates your focus and impact scores.
  • ❌ Cancel Task Stops the current task without saving any progress for it.

📊 Navigation

  • 📊 View Results Opens your results page showing all completed tasks, focus scores, impact scores, and total training time.
  • 🏠 Home Returns to this page so a different student can log in.

⚙️ Managing Tasks

  • ✏️ Edit Edit a task's title, description, estimated time, or difficulty.
  • 🧹 Clear Session Clears only the currently active task session. Your task list and completion history are kept.
  • 🔄 Reset Session Cancels the active task session and reloads the page. Keeps your task list and history.
  • 🗑️ Clear All Data for User Permanently deletes the student account, all tasks, and all history. Cannot be undone.

📈 Scores Explained

  • Focus Score How closely your actual time matched the estimated time. 1.0 is perfect; finishing early or running significantly over both lower your score.
  • Impact Score A combined score (0–10) based on task difficulty and your focus score. An Expert task done with perfect focus scores 10; an Easy task maxes at 2.5.
  • Difficulty Set manually in your CSV or via the Edit form. Four levels: 🟢 Easy (weight 0.5), 🟡 Medium (weight 1.0), 🟠 Hard (weight 1.5), 🔴 Expert (weight 2.0). If left blank, difficulty is auto-assigned by estimated time: under 5 min → Easy, 5–15 min → Medium, 15–30 min → Hard, over 30 min → Expert.
